的用户收取预订价。如果预订价高于正式发布当天的价格，将按照正式发布当天的价格向预订用户收费。 可以只在某些地区提供预订吗？ 取决于你的设置，默认为全选状态。如果实在不想在某个地区提供预订，可以选择不勾选该地区。需要注意，正式发布之后，你将无法在其他地区提供预订。 哪些用户可以预订 APP？ 预订仅提供给设备系统为 iOS 11.2, tvOS 11.2, and macOS 10.13.2
